# Claude Playground

A collection of single-file HTML tools built with Claude. Each tool is self-contained with zero dependencies -- just open in a browser.

## Tools

### [Write Review Playground](write-review-playground/)

A track-changes interface for reviewing prose edits. Paste original and edited text, review a word-level diff, and accept or reject each change individually. Includes keyboard shortcuts for fast reviewing and a live final text preview.

[Read more](write-review-playground/WRITE-REVIEW-PLAYGROUND.md)
